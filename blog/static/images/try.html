I like to keep things short: I assume readers know about GDCM; I will only show a quick, dirty way to create a DICOMDIR file, and hopefully illustrate the relevant portions of the 3 main header files:
GDCM::Directory
GDCM::DICOMDIRGenerator
GDCM::Writer
In essence, to sucessfully create the DICOMDIR, you need to pass the root directory containing DICOM files to the GDCM::DICOMDIRGenerator object as a string, set the appropriate flags (coming below), and call GDCM::DICOMDIRGenerator.Generate().
